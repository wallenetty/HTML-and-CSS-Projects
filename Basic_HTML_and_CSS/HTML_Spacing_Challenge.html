Using the copyright symbol

The copyright symbol can be written by it's Entity Name or by it's Entity Number. So if I write a sentence using the Entity Name code "&-c-o-p-y-;", it should appear as the © symbol. Or you can write a sentence using the Entity Number "&-#-1-6-9-;", it should appear as the © symbol also.
